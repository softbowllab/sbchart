## global.padding

`#2.0.0`

---

차트의 내부 컨텐츠의 padding을 지정합니다.

CSS padding을 사용하실 경우 차트가 그려질 Element는 padding을 0으로 지정하고 `global.padding`옵션을 사용하여 padding 값을 조정하는 것이 좋습니다.

* Type : object

* Format
```javascript
global: {
    padding: { 
        top:10, right:10, bottom:10, left:10
    }
}
```
