## axis.x.categories

`#2.0.0`

---

X축 카테고리를 지정합니다.

`axis.x.type="category"` 로 지정되어 있어야 합니다.

* Type : string[]

* Format
```javascript
axis: {
    x : {
        type: "category",
        categories: ["전국","서울","부산","대구","인천","광주","대전","울산","세종","경기","강원","충북","충남","전북","전남","경북","경남","제주"]
    }
}
```
