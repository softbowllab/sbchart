## global.size

_`#2.0.0`_

---

차트의 높이와 너비를 지정합니다.

이 옵션을 지정하지 않으면 첨부 된 상위 요소의 크기에 따라 차트 너비가 계산됩니다.

* Type : object

* Format
```javascript
global: {
    size: { 
        width:800, height:400
    }
}
```

### global.size.width 

_`#2.0.0`_

차트 너비를 지정 지정합니다.

* Type : number

* Default : 0

### global.size.height

_`#2.0.0`_

차트 높이를 지정 지정합니다.

* Type : number

* Default : 0
