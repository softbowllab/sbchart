## global.export

`#2.0.19`

---

차트를 이미지로 다운로드 하는 기능을 설정합니다.


* Type : Object

* Format
```javascript
global: {
	export: {
		show: true
	}
}
```

### global.export.show

`#2.0.19`

이미지 다운로드 기능의 사용 여부를 설정합니다. 기본값은 false입니다.

* Type : Boolean

* Default : false
