## legend.item
---

각 범례 세부 사항을  정의합니다. `#2.0.0`

* Type : object

* Format
```javascript
legend: {
    item: {
        tile: {
            height: 10,
            width: 10
        }
    }
}
```

### legend.item.tile.height

각 범례의 Tile 박스의 높이를 지정합니다. `#2.0.0`

* Type : number

* Default : 10


### legend.item.tile.width

각 범례의 Tile 박스의 너비를 지정합니다. `#2.0.0`

* Type : number

* Default : 10
