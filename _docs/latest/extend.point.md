## extend.point

`#2.0.0`

---

라인 차트 계열에서 포인트에 대한 확장 옵션을 지정합니다.

* Type : Object

* Format
```javascript
extend: {
	point: {
		show: true,
		r: 2.5
	}
}
```

### extend.point.show

`#2.0.0`

각 포인트의 show/hide 여부를 지정합니다.

* Type : Boolean

* Default : true


### extend.point.r

`#2.0.0`

각 포인트의 반지름 값을 지정합니다.

* Type : Number

* Default : 2.5
