## extend.point
---

점(포인트)에 대한 확장 옵션를 지정합니다.
`#2.0.0`

* Type : object

* Format
```javascript
extend: {
    point: {
        show: true,
        r: 2.5
    }
}
```

### extend.point.show

각 점을 줄에 표시할지 여부 지정합니다.
`#2.0.0`

* Type : boolean

* Default : true


### extend.point.r

각 점의 반지름값을 지정합니다.
`#2.0.0`

* Type : number

* Default : 2.5
