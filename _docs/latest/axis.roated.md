## axis.rotated
---

값이 `true` 이면 x, y축을 회전하여 가로 차트로 변경됩니다.

막대차트 계열만 지원되며("bar","stackbar" 등) 그 외 차트는 지원하지 않습니다.
`#2.0.0`

* Type : boolean

* Default : false

* Format
```javascript
axis: {
	rotated: true
}
```
