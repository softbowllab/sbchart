## data.legends

`#2.0.0`

---

범례 별로 라벨을 커스터마이징 하는 옵션입니다.

이 옵션을 지정하지 않으면 기본적으로 범례 아이디가 범례 라벨이 됩니다.

'범례 아이디': '범례 라벨' 구조입니다.

* Type : Object

* Format
```javascript
data: {
	legends: {'2015': '15년', '2016': '16년', '2017': '17년'}
}
```

예제에서 '2015', '2016', '2017' 은 범례 아이디

'15년', '16년', '17년' 은 범례 라벨입니다.
