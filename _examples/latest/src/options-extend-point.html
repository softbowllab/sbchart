포인트가 표시되는 차트에서 포인트의 크기(지름)를 변경합니다.

Options

  • extend.donut.title

    포인트의 지름을 지정합니다.

Example

Code

{% highlight javascript %} var chart = sb.chart.render("#chartWrap", { data : { columns: [ ["거래액", 7707750, 7573752, 7914913, 7386713, 8708523, 8624841, 8785755, 8048044, 9083890] ] }, extend: { point: {r:2.5} } }); document.getElementById("txtR").onchange = function() { chart.extend({point: {r: this.value} }).render(); }; {% endhighlight %}
