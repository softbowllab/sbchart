차트 영역과 X, Y축 사이의 여백(Padding)을 지정합니다. 이때 단위는 tick 단위입니다.

Options

X 축
  • axis.x.padding.left

    좌측 Tick단위 Padding을 지정합니다.

  • axis.x.padding.right

    우측 Tick단위 Padding을 지정합니다.

Y 축
  • axis.y.padding.top

    상단 Tick단위 Padding을 지정합니다.

  • axis.y.padding.bottom

    하단 Tick단위 Padding을 지정합니다.

Example

Code

{% highlight javascript %} var chart = sb.chart.render("#chartWrap", { data: { type: "bar", columns: [ ["2017",1.48,3.64,2.35,1.29,1.42,1.34,1.51,-1.08,4.29,1.67,2.4,-0.36,-0.53,1.98,2.23,-0.9,-1.62,1.66] ] }, axis: { x: { padding: { left: 1, right: 3 } }, y: { padding: { top: 1, bottom: 1 } } } }); document.getElementById("txtLeft").value = chart.axis().x.padding.left; document.getElementById("txtRight").value = chart.axis().x.padding.right; document.getElementById("txtTop").value = chart.axis().y.padding.top; document.getElementById("txtBottom").value = chart.axis().y.padding.bottom; document.getElementById("txtLeft").onchange = function() { chart.axis({ x: { padding: { left: this.value } } }).render(); }; document.getElementById("txtRight").onchange = function() { chart.axis({ x: { padding: { right: this.value } } }).render(); }; document.getElementById("txtTop").onchange = function() { chart.axis().y.padding.top = this.value; chart.render(); }; document.getElementById("txtBottom").onchange = function() { chart.axis().y.padding.bottom = this.value; chart.render(); }; {% endhighlight %}
