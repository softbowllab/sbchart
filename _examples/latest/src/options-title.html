타이틀을 지정하고 표시 위치를 변경합니다.

Options

  • title.text

    타이틀을 지정합니다.

  • title.position

    타이틀 위치를 지정합니다.

Padding
  • title.padding.top

    상단 Padding을 지정합니다.

  • title.padding.right

    우측 Padding을 지정합니다.

  • title.padding.bottom

    하단 Padding을 지정합니다.

  • title.padding.left

    좌측 Padding을 지정합니다.
