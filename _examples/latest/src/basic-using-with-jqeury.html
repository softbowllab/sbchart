jquery.js 파일이 sbchart.js 파일보다 먼저 선언되어 있다면 SBChart를 jquery 플러그인으로 사용할 수 있습니다.

$("..").sbChart();형태로 사용하며
파라미터(옵션)는 일반적인 javascript 사용 방식과 동일합니다.

Format

{% highlight javascript %} $("....").sbChart({data:{ ... }}); {% endhighlight %}
