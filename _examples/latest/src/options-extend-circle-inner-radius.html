도넛/게이지(donut/gauge) 차트의 내부 반지름을 지정합니다.

Options

  • extend.[donut/gauge].innerRadius

    내부 반지름을 지정합니다.

Example

Code

{% highlight javascript %} //donut var chart1 = sb.chart.render("#chartWrap1", { data: { type: "donut", columns: [ ["2015",3.51], ["2016",0.71], ["2017",1.48] ] } }); //gauge var chart2 = sb.chart.render("#chartWrap2", { data: { type: "gauge", columns: [ ["2015",71] ] } }); // innerRadius document.getElementById("txtInnerRadius").onchange = function() { //donut chart1.extend({ donut: { innerRadius: this.value } }).render(); //gauge chart2.extend({ gauge: { innerRadius: this.value } }).render(); }; {% endhighlight %}
