axis.x.type = "category" 혹은 axis.x.type = "timeseries"일 때 사용합니다.
X축의 카테고리를 지정하고 Tick에 표시될 문자열 혹은 시간을 정의합니다.

axis.x.type = "category"일 때는 Tick이 X축 영역 사이에 표시됩니다.

Options

  • axis.x.type

    axis.x.categories = ["전국","서울","부산",...,"경북","경남","제주"]

  • axis.x.categories = ["2019-01-01","2019-01-02","2019-01-03",...,"2019-01-16","2019-01-17","2019-01-18"]
