레이더 차트 기본 예제입니다. data.type="radar"로 지정합니다.

data.x를 사용하여 카테고리명을 설정하여 사용합니다.

Example

Code

{% highlight javascript %} sb.chart.render("#chartWrap", { data: { type: "radar", x: "x", columns: [ ["x", "Data A", "Data B", "Data C", "Data D", "Data E"], ["data1", 330, 350, 200, 380, 150], ["data2", 130, 100, 30, 200, 80], ["data3", 230, 153, 85, 300, 250] ] } }); {% endhighlight %}
