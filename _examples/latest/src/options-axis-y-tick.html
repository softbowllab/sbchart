--- title: Axis-Y Tick date: 2018-06-04 categories: ["latest","options"] order: 334 ---

Y축의 tick을 커스트마이징 하거나 표시될 tick을 선택 할 수 있습니다.

Example

axis.y.tick.format=function(d){return (""+Math.floor(d/1000)).replace(/\B(?=(\d{3})+(?!\d))/g, ",")+" 천만";}
axis.y.tick.values = [7000000, 8000000, 9000000]

Code

{% highlight javascript %} var chart = sb.chart.render("#chartWrap", { data : { type: "bar", columns: [ ["온라인쇼핑몰 거래액", 7707750, 7573752, 7914913, 7386713, 8708523, 8624841, 8785755, 8048044, 9083890] ] }, axis: { y: { tick: { format: function(d) { return ("" + Math.floor(d/1000)).replace(/\B(?=(\d{3})+(?!\d))/g, ",") + " 천만"; }, values:[7000000, 8000000, 9000000] } } } }); {% endhighlight %}
