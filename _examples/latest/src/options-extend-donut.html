--- title: Donut Options date: 2018-07-26 categories: ["latest","options"] order: 426 --- 도넛(donut) 차트 고유 옵션입니다.

Options

  • extend.donut.title

    내부 원 안에 타이틀을 지정합니다.

Example

Code

{% highlight css %} #chartWrap .sbchart-arcs-title { font-size: 2em; } /* 폰트사이즈 */ {% endhighlight %}

{% highlight javascript %} var chart = sb.chart.render("#chartWrap", { data: { type: "donut", columns: [ ["2015",3.51], ["2016",0.71], ["2017",1.48] ] }, extend: { donut: { title: document.getElementById("txtTitle").value } } }); // title document.getElementById("txtTitle").onchange = function() { chart.extend({ donut: { title: this.value } }).render(); }; {% endhighlight %}
