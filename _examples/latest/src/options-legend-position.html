범례 위치를 오른쪽, 아래쪽 또는 차트 내부(inset)로 이동할 수 있습니다.
차트내부(inset) 로 위치를 지정 하였을 경우는 세부위치 조정이 가능합니다.

Options

  • legend.position

    범례의 위치를 지정합니다.

  • legend.background

inset (legend.position="inset")
  • legend.inset.anchor

    범례 내부 위치를 지정합니다.

  • legend.inset.x

    X축으로 값만큼 위치를 조정합니다.

  • legend.inset.y

    Y축으로 값만큼 위치를 조정합니다.

  • legend.inset.step

    범례의 행수를 지정합니다.

Example

{% highlight javascript %} var chart = sb.chart.render("#chartWrap", { data : { type: "bar", columns: [ ["2015",3.51,4.6,3.43,7.96,3.37,5.83,0.35,3.23,-0.09,4.47,2.21,1.13,0.77,0.35,0.92,2.65,1.64,8.08], ["2016",0.71,2.14,3.18,-1.84,0.79,0.35,0.28,0.44,0.79,0.84,1.33,-0.7,-1.51,0.09,1.05,-1.66,-0.58,4.63], ["2017",1.48,3.64,2.35,1.29,1.42,1.34,1.51,-1.08,4.29,1.67,2.4,-0.36,-0.53,1.98,2.23,-0.9,-1.62,1.66] ] }, legend: { position: "inset", item: { tile: {height:5, width:2} }, inset: { anchor: "top-left", x: 180, y: 20, step: 2 } } }); {% endhighlight %}
