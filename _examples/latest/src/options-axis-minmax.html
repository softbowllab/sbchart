--- title: Axis Min/Max date: 2018-06-05 categories: ["latest","options"] order: 332 ---

데이터 값에 따라 X, Y축의 min, max 값이 자동 계산됩니다.
자동 계산되는 min, max 값 대신 임의의 Min 값과 Max 값을 정의할 수 있습니다.

Options

X 축
  • axis.x.min

    최솟값을 지정합니다.

  • axis.x.max

    최댓값을 지정합니다.

Y 축
  • axis.y.min

    최솟값을 지정합니다.

  • axis.y.max

    최댓값을 지정합니다.

Example

Code

{% highlight javascript %} var chart = sb.chart.render("#chartWrap", { data: { type: "bar", columns: [ ["2017",1.48,3.64,2.35,1.29,1.42,1.34,1.51,-1.08,4.29,1.67,2.4,-0.36,-0.53,1.98,2.23,-0.9,-1.62,1.66] ] }, axis: { x: { padding: 0, min: -1, max: 19 }, y: { padding: 0, min: -4, max: 6 } } }); document.getElementById("txtXMin").value = chart.axis().x.min; document.getElementById("txtXMax").value = chart.axis().x.max; document.getElementById("txtYMin").value = chart.axis().y.min; document.getElementById("txtYMax").value = chart.axis().y.max; document.getElementById("txtXMin").onchange = function() { chart.axis({ x: { min: this.value } }).render(); }; document.getElementById("txtXMax").onchange = function() { chart.axis({ x: { max: this.value } }).render(); }; document.getElementById("txtYMin").onchange = function() { chart.axis({ y: { min: this.value } }).render(); }; document.getElementById("txtYMax").onchange = function() { chart.axis({ y: { max: this.value } }).render(); {% endhighlight %}
