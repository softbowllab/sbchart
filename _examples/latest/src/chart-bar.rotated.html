막대 차트 가로 그리기 예제입니다.

data.type="bar" 로 그리고 "axis.rotated="true"로 지정합니다.

Example

Code

{% highlight javascript %} var conf = { global : { size : { width: 800, height: 320 } }, data : { type: "bar", columns: [ ["2017",2.8,1.45,2.35,0.29,0.42,2.0,5.12,1.12,2.3,2.67] ] }, legend: {show: false}, axis: {rotated: true} }; SBChart.render("#chartWrap", conf); {% endhighlight %}
