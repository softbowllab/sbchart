--- title: Size date: 2018-06-12 categories: ["latest","options"] order: 311 ---

차트의 크기를 지정합니다.
크기를 따로 지정하지 않는다면 차트가 그려지는 영역(컨테이너) 크기에 맞춰집니다.

Options

  • global.size.width

    차트 너비를 지정합니다.

  • global.size.height

    차트 높이를 지정합니다.

Example

Code

{% highlight javascript %} var chart = sb.chart.render("#chartWrap", { data: { type: "bar", columns: [ ["2017",1.48,3.64,2.35,1.29,1.42,1.34,1.51,-1.08,4.29,1.67,2.4,-0.36,-0.53,1.98,2.23,-0.9,-1.62,1.66] ] } }); //width document.getElementById("txtWidth").onchange = function() { chart.global({ size: { width: this.value } }).render(); }; //height document.getElementById("txtHeight").onchange = function() { chart.global({ size: { height: this.value } }).render(); }; {% endhighlight %}
