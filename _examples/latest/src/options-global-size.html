--- title: Size date: 2018-06-12 categories: ["latest","options"] order: 311 ---

차트의 크기 및 여백을 지정합니다.
크기를 따로 지정하지 않는다면 차트가 그려지는 영역(컨테이너) 크기에 맞춰집니다.

Options

  • global.size.width

    차트 너비를 지정합니다.

  • global.size.height

    차트 높이를 지정합니다.

  • global.padding.top

    차트 상단 Padding을 지정합니다.

  • global.padding.bottom

    차트 하단 Padding을 지정합니다.

  • global.padding.left

    차트 좌측 Padding을 지정합니다.

  • global.padding.right

    차트 우측 Padding을 지정합니다.
