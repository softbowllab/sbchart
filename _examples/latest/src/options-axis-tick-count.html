--- title: Axis Tick Count date: 2018-06-04 categories: ["latest","options"] order: 334 ---

X, Y축의 Tick 개수를 지정합니다.

Options

  • axis.x.tick.count

    X축 Tick 개수를 지정합니다. X축 모두를 표현하려면 0을 입력합니다.

  • axis.y.tick.count

    Y축 Tick 개수를 지정합니다.

{% highlight javascript %} var chart = sb.chart.render("#chartWrap", { data: { type: "bar", columns: [ ["2016",0.71,2.14,3.18,-1.84,0.79,0.35,0.28,0.44,0.79,0.84,1.33,-0.7,-1.51,0.09,1.05,-1.66,-0.58,4.63], ["2017",1.48,3.64,2.35,1.29,1.42,1.34,1.51,-1.08,4.29,1.67,2.4,-0.36,-0.53,1.98,2.23,-0.9,-1.62,1.66] ] }, axis: { x: { tick: { count: 5 } }, y: { tick: { count: 4 } } } }); //count-x document.getElementById("txtX").onchange = function() { chart.axis({ x: { tick: { count: this.value } } }).render(); }; //count-y document.getElementById("txtY").onchange = function() { chart.axis({ y: { tick: { count: this.value } } }).render(); }; {% endhighlight %}
