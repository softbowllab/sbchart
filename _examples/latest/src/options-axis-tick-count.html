--- title: Axis Tick Count date: 2018-06-04 categories: ["latest","options"] order: 334 ---

X, Y축의 Tick 갯수를 지정합니다.
X축은 2이하의 값은 유효하지 않습니다.

Example

X축
axis.x.tick.count=10
Y축
axis.y.tick.count=10

Code

{% highlight javascript %} var chart = sb.chart.render("#chartWrap", { data : { type: "bar", columns: [ ["2017",1.48,3.64,2.35,1.29,1.42,1.34,1.51,-1.08,4.29,1.67,2.4,-0.36,-0.53,1.98,2.23,-0.9,-1.62,1.66] ] }, axis: { x: { tick: { count: 5 } }, y: { tick: { count: 4 } } } }); {% endhighlight %}
