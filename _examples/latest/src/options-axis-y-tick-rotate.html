--- title: Tick Rotation (Rotated) date: 2018-08-16 categories: ["latest","options"] order: 339 ---

X축의 Tick을 지정된 각도로 회전하거나 멀티라인으로 표시합니다.
가로 차트axis.rotated = true인 경우 X, Y축이 서로 바뀌므로 X축의 Tick을 회전하기 위해서는 Y축 옵션을 사용합니다.

Options

  • axis.y.tick.rotate

    Tick을 해당 각도로 회전합니다.

Code

{% highlight javascript %} var chart = sb.chart.render("#chartWrap1", { data: { type: "bar", columns: [ ["2016", 7707750, 7573752, 7914913, 7386713, 8708523, 8624841, 8785755, 8048044, 9083890], ["2017", 7958450, 8392392, 9011231, 8092838, 8829832, 7029282, 8103392, 7403013, 7182900] ] }, axis: { rotated: true, y: { tick: { rotate: 45 } } } }); //rotate document.getElementById("txtRotateY").onchange = function() { chart.axis({ y: { tick: { rotate: this.value } } }).render(); }; {% endhighlight %}
