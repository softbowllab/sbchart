--- title: Axis-Y Tick Rotate date: 2018-08-16 categories: ["latest","options"] order: 339 ---

세로 차트axis.rotated == true인 경우는 X,Y 축이 서로 바뀌므로
뒤바뀐 X축의 Tick 지정된 각도를 회전하기 위해서는 Y축 옵션을 사용합니다.

Example

axis.y.tick.rotate=45

Code

{% highlight javascript %} var chart1 = sb.chart.render("#chartWrap1", { data : { type: "bar", columns: [ ["2017",1.45,2.35,0.29,0.42,2.0,5.12,1.12,2.3,2.66] ] }, axis: { y: { tick: { rotate: 45 } } } }); {% endhighlight %}
