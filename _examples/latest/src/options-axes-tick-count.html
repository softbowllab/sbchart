Y축의 tick 라벨을 변경하거나 display 될 tick을 선택 할 수 있습니다.

X축은 2이하의 값은 유효하지 않습니다.


X축
axes.x.tick.count=10
Y축
axes.y.tick.count=10

Code

{% highlight javascript %} var conf = { global : { size : { width: 800, height: 320 } }, data : { type: "bar", columns: [ ["2017",1.48,3.64,2.35,1.29,1.42,1.34,1.51,-1.08,4.29,1.67,2.4,-0.36,-0.53,1.98,2.23,-0.9,-1.62,1.66] ] }, axes: { x: { tick: { count: 5 } }, y: { tick: { count: 4 } } } }; SBChart.render("#chartWrap", conf); {% endhighlight %}
