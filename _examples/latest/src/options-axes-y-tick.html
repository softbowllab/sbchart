Y축의 tick 라벨을 변경하거나 display 될 tick을 선택 할 수 있습니다.

format
(예) 라벨 변경
values
(예) display 값 선택

Code

{% highlight javascript %} var conf = { global : { size : { width: 800, height: 320 } }, data : { type: "bar", columns: [ ["온라인쇼핑몰 거래액", 7707750, 7573752, 7914913, 7386713, 8708523, 8624841, 8785755, 8048044, 9083890] ] }, axes: { y: { tick: { format: function(d) { return ("" + Math.floor(d/1000)).replace(/\B(?=(\d{3})+(?!\d))/g, ",") + " 천만"; }, values:[7000000, 8000000, 9000000] } } } }; SBChart.render("#chartWrap", conf); {% endhighlight %}
