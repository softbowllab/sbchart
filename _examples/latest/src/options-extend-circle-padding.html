원/도넛(pie/donut) 차트의 영역 사이 여백(Padding)을 지정합니다.

Options

  • extend.[pie/donut].padding

    영역 사이 여백(Padding)을 지정합니다.

Example

Code

{% highlight javascript %} //pie var chart1 = sb.chart.render("#chartWrap1", { data: { type: "pie", columns: [ ["2015",3.51], ["2016",0.71], ["2017",1.48] ] } }); //donut var chart2 = sb.chart.render("#chartWrap2", { data: { type: "donut", columns: [ ["2015",3.51], ["2016",0.71], ["2017",1.48] ] } }); // padding document.getElementById("txtPadding").onchange = function() { //pie chart1.extend({ pie: { padding: this.value } }).render(); //donut chart2.extend({ donut: { padding: this.value } }).render(); }; {% endhighlight %}
