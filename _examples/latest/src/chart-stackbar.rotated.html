Stacked Bar 차트 가로 처리 기본 예제입니다. data.type="stackbar" 로 그리고 axis.rotated="true"로 지정합니다.

Example

Code

{% highlight javascript %} sb.chart.render("#chartWrap", { data: { type: "stackbar", columns: [ ["2015",3.51,4.6,3.43,7.96,3.37,5.83], ["2016",0.71,2.14,3.18,-1.84,0.79,0.35], ["2017",1.48,3.64,2.35,1.29,1.42,-0.34] ] }, axis: { rotated: true } }); {% endhighlight %}
